# Desarrollo Web en Entorno Cliente (DWC)

Este repositorio contiene mis ejercicios y proyectos realizados para el módulo profesional de **Desarrollo Web en Entorno Cliente**.

El contenido está enfocado principalmente en las siguientes tecnologías:

- **JavaScript** (Vanilla JS y ES6+)
- **React**
- **TypeScript**

Aquí iré subiendo el progreso de las diferentes unidades y prácticas del curso.
