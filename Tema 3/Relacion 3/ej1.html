Diseña una web que cree un Array llamado clase que contenga información de una clase. Cada elemento del array debe ser una tupla de 4 campos: nombre, edad, nota primer trimestre, nota segundo trimestre y nota tercer trimestre (todos separados por comas). Por ejemplo: clase[0] = “Angel Garcia, 20, 6, 7, 10”
