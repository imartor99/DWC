Crea una página web que contenga varios párrafos con texto de ejemplo. La página contendrá un botón que al pulsarlo solicitará al usuario una palabra o cadena de texto e indicará si existe o no dicha palabra dentro del document

Búsqueda de Texto en Párrafos (Resultado con document.write)

Este es el primer párrafo con texto de ejemplo. La programación funcional en JavaScript se centra en qué hacer, evitando bucles explícitos al trabajar con colecciones.

Para manipular colecciones de elementos del DOM, es esencial convertirlas a un array usando Array.from.

La propiedad length de un array no tiene tamaño fijo, ya que los arrays son un tipo de objeto en JavaScript.

Usaremos el método includes() para comprobar la existencia de la palabra dentro de la cadena completa.
