Control de Estilos de Texto (Ejercicio 2)

Párrafo 1. La manipulación del Modelo de Objetos del Documento (DOM) mediante JavaScript es fundamental para crear páginas web interactivas y accesibles. En este ejercicio, las funciones alteran las propiedades CSS de los elementos de texto en tiempo real, mejorando la experiencia del usuario sin recargar la página.

Párrafo 2. El control del tamaño de la fuente se realiza obteniendo el valor actual de la propiedad 'font-size', modificándolo con un incremento/decremento fijo y reasignando el nuevo valor. Es crucial usar unidades relativas como `em` o `rem` para asegurar que el escalado sea proporcional.

Párrafo 3. La capacidad de ajustar el tamaño del texto es una característica clave de accesibilidad. Permite a usuarios con distintas necesidades visuales adaptar el contenido a sus preferencias específicas. El botón de 'Tamaño por Defecto' asegura que siempre se pueda volver a una configuración base estándar.

Párrafo 4. La alineación del texto se controla mediante la propiedad 'text-align'. Utilizamos funciones simples para alternar entre 'justify' (texto justificado a ambos lados) y 'left' (alineación a la izquierda). Cada botón llama directamente a la función `cambiarAlineacion()` con el parámetro deseado.

Párrafo 5. Siguiendo las instrucciones, cada funcionalidad (aumento, decremento, reseteo, alineación) está encapsulada en una función de JavaScript independiente. Esto hace que la página sea fácil de extender y depurar.
